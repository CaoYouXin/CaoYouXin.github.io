#### 坑的分布

一句话：由于我不喜欢使用太多第三方类库，所以最坑的就是自己写的新功能了。包含有：

1. ul>li>iframe，主页面结构
2. zoom，实际页面都会有缩放，PC端正常显示除外。
3. history stack，一般业务开发中很少接触，实现了炫酷的功能。
4. 原生API事件绑定，移动端touch,click绑定两次。设置400ms事件间隔。（不知道动画中会不会是个巨坑啊）
